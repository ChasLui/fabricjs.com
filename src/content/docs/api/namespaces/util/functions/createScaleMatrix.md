> **createScaleMatrix**(`x`, `y`?): [`TMat2D`](../../../type-aliases/TMat2D.md)

Generate a scale matrix around the point (0,0)

A matrix in the form of
[x 0 0]
[0 y 0]
[0 0 1]

## Parameters

• **x**: `number`

scale on X axis

• **y?**: `number`= `x`

scale on Y axis

## Returns

[`TMat2D`](../../../type-aliases/TMat2D.md)

matrix
